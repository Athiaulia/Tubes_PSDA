## Deskripsi

Aplikasi **Manajemen Member Gym** berbasis GUI (Graphical User Interface) yang dibangun menggunakan Python dan PySide6. Aplikasi ini mengelola data keanggotaan gym dengan mengimplementasikan struktur data **Doubly Linked List (DLL)**.

Data yang dikelola:
- ID Member
- Nama Member
- Paket Langganan (Basic / Silver / Gold)
- Sisa Durasi (hari)

**Logika Khusus:** Fitur sorting yang mengurutkan data berdasarkan sisa masa aktif member dari yang paling dekat expired (terkecil ke terbesar).

---

## Fitur

| Fitur | Keterangan |
|-------|------------|
| **Insert** | Menambahkan data member baru (ID unik, validasi otomatis) |
| **Delete** | Menghapus data member berdasarkan ID |
| **Search** | Mencari member berdasarkan ID |
| **Sort** | Mengurutkan member berdasarkan sisa durasi (paling dekat expired) |
| **Display** | Menampilkan seluruh data member di tabel |
| **Clear** | Membersihkan form input |
